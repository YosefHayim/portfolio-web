# Portfolio Web

This repository contains the source code for my personal portfolio website. It serves as a central hub to present my professional profile, technical skills, and selected projects in a structured and modern layout.

## Overview

The portfolio is designed to provide a clear picture of who I am as a software engineer. It includes dedicated sections for my experience, tech stack, projects, and ways to contact me. The site reflects my growth, work style, and approach to building software.

## Features

- Clean and modern UI.
- Fully responsive design.
- Sections for skills, experience, and project showcases.
- Contact and social links for easy outreach.
- Built to be easily extendable as my career evolves.

## Tech Stack

While the implementation uses modern web tools, the portfolio itself focuses on content and presentation rather than any specific framework.  
Technologies used include:

- HTML  
- CSS / TailwindCSS  
- JavaScript / TypeScript  
- React  
- Vite  
- UI utilities such as Shadcn
